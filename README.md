# Airbnb-Host-Response-Analytics
Project Overview

This project focuses on analyzing Airbnb listings in Paris to understand property performance, pricing trends, host behavior, and customer review patterns. The dashboard was developed using Power BI, DAX, Python Visuals, and Data Modeling techniques to transform raw Airbnb data into meaningful business insights.
The solution enables stakeholders to evaluate listing quality, identify high-performing hosts, understand pricing strategies, and monitor customer satisfaction through an interactive and visually appealing dashboard.

Project Objective

The primary objectives of this project are:
Analyze Airbnb listings across Paris.
Understand host performance and responsiveness.
Evaluate property pricing and accommodation trends.
Measure guest satisfaction through review analytics.
Identify factors influencing ratings and pricing.
Build an interactive dashboard for business decision-making.

Key Insights

Most Airbnb listings are concentrated within a few popular property types.
Super Hosts demonstrate higher response and acceptance rates.
Listings accommodating more guests generally command higher prices.
Property type significantly influences pricing.
Guest ratings are strongly influenced by cleanliness, communication, and check-in experience.
A small number of listings generate a large share of reviews.
Higher-rated listings tend to maintain consistent guest experience across review dimensions.

Conclusion

The Airbnb Paris Listings & Host Performance Analytics Dashboard successfully transforms raw listing and review data into actionable business intelligence. The dashboard provides a comprehensive view of property performance, pricing trends, host effectiveness, and customer satisfaction, enabling data-driven decision-making for stakeholders and supporting strategic improvements in Airbnb operations.
